# My Prose App

## Overview

My Prose App is a Java Maven application that generates prose by combining multiple sentence implementations. It uses the `Sentence` interface and `ProseBuilder` library.

## Features

- Multiple `Sentence` implementations
- Uses Maven dependency management
- Unit tests with JUnit
- GitHub Actions for CI
- GitHub Packages deployment

## Requirements

- Java 21
- Maven 3.x

## Build

```bash
mvn clean install
```

## Run

```bash
mvn exec:java
```

## Tests

```bash
mvn test
```
